## 🛠️ **Quick Start Guide**

### **Prerequisites**
```bash
node --version  # ≥18.0.0
npm --version   # ≥9.0.0
```

### **Installation & Setup**

```bash
# 1️⃣ Clone the treasure
git clone https://github.com/albertomarinmorales-coder/learningReact-TS.git

# 2️⃣ Navigate to your ship
cd learningReact-TS

# 3️⃣ Install your crew (dependencies)
npm install

# 4️⃣ Set sail (start development server)
npm run dev

# 🎉 Open http://localhost:5173 and start your adventure!
```

### **Available Commands**

| Command | Description | When to Use |
|---------|-------------|-------------|
| `npm run dev` | Start development server | During active development |
| `npm run build` | Create production build | Before deployment |
| `npm run preview` | Preview production build | Testing before release |
| `npm run lint` | Run ESLint checks | Code quality assurance |
